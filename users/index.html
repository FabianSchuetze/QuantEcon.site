Python

Installing the quantecon Python Package

The core of the QuantEcon Python code library is the Python package quantecon

Assuming you have pip on your computer you can install the latest stable release of quantecon by typing

pip install quantecon

at a terminal prompt

For more detailed instructions see this page.

Downloading the Repository

The source of the quantecon package and additional examples can be found in the Python code repository.

You can either download the zip file directly or Use Git to clone it. To learn the basics about setting up Git see here

Julia

Install QuantEcon.jl

To install the Julia QuantEcon package QuantEcon.jl start a Julia session and type

Pkg.add("QuantEcon")

More detailed instructions are available at quant-econ.net.
