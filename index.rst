QuantEcon
=========

QuantEcon is an international organization supporting distributed development of high quality open source code for quantitative economic modeling.  QuantEcon is supported by the Alfred P. Sloan Foundation and coordinated by economists for the benefit of researchers in economics, finance and other related fields.  The main resources provided by QuantEcon are peer reviewed high performance code libraries written in Python and Julia.
